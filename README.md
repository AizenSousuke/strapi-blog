# Next + Strapi Blog

Following tutorial from [https://strapi.io/blog/build-a-blog-with-next-react-js-strapi](https://strapi.io/blog/build-a-blog-with-next-react-js-strapi)

## Getting started
```
cd backend
yarn develop
cd ..
cd frontend
yarn dev
```

Go to http://localhost:3000 in any browser
